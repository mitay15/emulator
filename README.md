# AAPS Emulator — Full AutoISF Algorithm Emulator + Auto‑GA v3 Optimizer

AAPS Emulator — это полностью воспроизводимый Python‑эмулятор алгоритма **AutoISF** из AndroidAPS, дополненный профессиональным **генетическим оптимизатором Auto‑GA v3**, который автоматически подбирает профиль под реальные логи AAPS.

Проект создан для:

- точного сравнения расчётов Python ↔ AndroidAPS  
- анализа поведения AutoISF на реальных логах  
- визуализации predBG, IOB, COB, UAM, ZT  
- автоматической проверки полного соответствия алгоритму  
- **автоматического подбора sens / carb_ratio / AutoISF параметров под реальные данные**  

---

# 🚀 Возможности

## 🔬 Эмуляция алгоритма AutoISF
- Полная реализация DetermineBasal + AutoISF pipeline  
- Поддержка всех внутренних переменных AutoISF  
- Полное соответствие Kotlin‑логике AndroidAPS  
- Поддержка autosens, UAM, SMB, ZT, IOB, COB  
- Полная совместимость с логами AAPS (Kotlin parser)

## 📈 Аналитика и визуализация
- eventualBG, minPredBG, minGuardBG  
- SMB, temp basal, insulinReq  
- autosens, variable_sens  
- predBG curves (IOB/COB/UAM/ZT)  
- интерактивные графики Plotly  
- HTML‑отчёты  
- тепловые карты расхождений  

## 🤖 Auto‑GA v3 — профессиональный оптимизатор профиля

Новый оптимизатор профиля включает:

### ✔ Адаптивная мутация  
- усиливается при застое  
- ослабляется при прогрессе  

### ✔ Адаптивный элитизм  
- увеличивается при шуме  
- уменьшается при стабильности  

### ✔ Адаптивный размер популяции  
- расширяется при отсутствии улучшений  
- сужается при стабильном прогрессе  

### ✔ Оценка разнообразия популяции  
- предотвращает «залипание» GA  
- добавляет случайные особи при необходимости  

### ✔ Адаптивные диапазоны параметров  
- расширяются, если оптимум у края  
- сужаются, если параметр стабилен  

### ✔ Мягкий, умный early stopping  
- останавливает оптимизацию, когда прогресс прекращается  
- экономит время без потери качества  

### ✔ Полная интеграция с GUI  
- отображение поколения  
- отображение fitness  
- отображение диагностической строки Auto‑GA
